🧠 EfficientNet-B0 骨干网络

鱼眼新鲜度评估是一个细粒度视觉分类任务，需要高精度且计算高效的神经网络来适配移动端部署。FishFreshNetV1 选用 EfficientNet-B0 作为基础架构，核心优势如下：

  • 复合缩放方法（Compound Scaling）[1]：EfficientNet 通过统一的复合系数 φ 同时缩放网络深度（depth）、宽度（width）和输入分辨率（resolution），三者协同优化而非单一维度扩展，在保持高精度的同时显著减少参数量。
  • MBConv 模块（Mobile Inverted Bottleneck Convolution）：作为 EfficientNet-B0 的基本构建单元，MBConv 采用深度可分离卷积 + 反残差结构（先升维→深度卷积→降维），在大幅降低计算量的同时保持高特征提取能力。
  • 自定义分类头：移除 EfficientNet-B0 原始分类头，替换为自适应平均池化层（Adaptive Average Pooling）→ Dropout 层 → 线性全连接层，有效缓解训练过拟合并映射到三个新鲜度类别（高度新鲜 / 新鲜 / 不新鲜）。

🎯 CBAM 注意力机制

在实际检测场景中，鱼眼图像常包含大量无关信息（泡沫箱、光照变化、鱼鳞等）。为使网络聚焦于关键鱼眼区域，FishFreshNetV1 在 EfficientNet-B0 最后一个特征提取层之后（输入通道数 1280）集成了 CBAM（Convolutional Block Attention Module）[2]。CBAM 是一个轻量级注意力模块，沿通道和空间两个维度依次推送注意力图：

通道注意力子模块

对特征图同时执行最大池化和平均池化，将两个池化结果输入共享的多层感知机（MLP）。MLP 通过先降维再升维的结构学习通道间依赖关系，输出通道注意力权重。

→ 帮助模型推断哪些特征通道真正具有判别力（如角膜浑浊度相关的特征通道）

↓ 特征图 × 通道权重
空间注意力子模块

沿通道轴执行池化操作（最大池化 + 平均池化），生成 2 通道的聚合特征，再通过卷积层生成空间注意力图。

→ 引导网络聚焦瞳孔和角膜区域，抑制背景环境（砧板、纸张、泡沫箱）的干扰

↓ 特征图 × 空间权重
自适应特征精炼

两张注意力图分别与输入特征图相乘，实现通道和空间维度的自适应特征精炼，在模型体积与识别精度之间取得平衡。

🔥 Grad-CAM 可解释性

为克服深度神经网络的"黑箱"问题，FishFreshNetV1 引入 Grad-CAM（Gradient-weighted Class Activation Mapping）[3] 可视化技术：

  • 利用目标类别梯度流入 EfficientNet-B0 最后一个卷积层，生成粗粒度定位图
  • 对梯度执行全局平均池化，计算每个特征通道的重要性权重
  • 生成热力图，高亮对模型预测影响最大的图像空间区域

论文实验表明：在正确分类的案例中，红色高激活区域精准聚焦于瞳孔和角膜；在错误分类的案例中，高激活区域偏移至眼周明亮鱼鳞或镜面反射光斑——这证实模型确实依据生物学特征做出判断，而非依赖背景捷径。

创作历程

💡

灵感诞生

陪家人去菜市场买鱼，摊主信誓旦旦"绝对新鲜"，回家切开却已发臭。发现鱼眼是反映新鲜度最关键的视觉信号，但人眼无法分辨细微差距——如果 AI 能看懂鱼眼呢？

🔬

独立研究

撰写论文《FishFreshNetV1: A Lightweight and Explainable Framework Based on Attention Mechanism for Fish Freshness Assessment》，提出 EfficientNet-B0 + CBAM 架构。构建 MFED 数据集（4800 张 / 4 环境 / GB 2733-2015 国标），达成 99.23% 准确率。数据集开源至 Mendeley Data。

⚙️

后端开发

FastAPI + PyTorch 模型推理服务，部署至 Hugging Face Spaces。LangGraph 16 节点智能体工作流，置信度自适应分支 + 多模态 LLM 兜底，确保每一次检测都有可靠结果。

🎨

前端开发（初赛）

TRAE Work 5 次对话从零到交付。项目奠基 → 视觉升级 → API 集成 → 结果深化 → 打磨交付。单文件 HTML 149KB，深海动态主题 + 4-Tab 分析报告 + 离线演示模式。

🚀

复赛迭代

多页面重构 → 移动端深度适配 + PWA → 16 节点工作流融合 → 置信度自适应前端机制 → 最终交付打磨。从"能用的 Demo"进化为"好用的产品"。
